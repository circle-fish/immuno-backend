# immuno-backend
免疫荧光流程优化项目
java后端设计一个子系统，与公司核心系统对接的同时，简化实验室的流程操作
，只提供必要的信息与结果展示，以及和其他例如提供AI分析系统的接口的调用，并储存对应调用结果，
整理所有信息以表单、图片等形式返回。
